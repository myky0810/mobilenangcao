# Hướng dẫn thêm hình ảnh xe

Để giao diện homescreen hiển thị đầy đủ, bạn cần thêm các hình ảnh xe vào thư mục:
`assets/images/products/`

## Danh sách file cần thêm:
- car1.jpg (Mercedes GLC)
- car2.jpg (BMW 7)
- car3.jpg (Acura)
- car4.jpg (Toyota Camry 2025)
- car5.jpg (KGM Utw 2025)
- car6.jpg (Corolla)

## Lưu ý:
- Nếu không có hình, app sẽ hiển thị icon xe placeholder (không bị crash)
- Hình nên có tỷ lệ ngang (landscape) để đẹp hơn
- Kích thước khuyến nghị: 800x600px hoặc tương tự
